  • OOPS Concepts in Java -


  • Object-oriented programming model revolves around the concept of Objects. What is an Object? An object is an instance of a Class. It contains properties and functions.They are like real-world objects. i.e. car, house, laptop are all objects. They have some particular properties and methods to perform some action. What is a Class? The Class defines the blueprint of Objects. They define the properties and functionalities of the objects. For example, Laptop is a class and your laptop is an instance of it.

    1. Abstraction

    3. Polymorphism
    4. Inheritance
