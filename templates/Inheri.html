  • Inheritance in Java -


  • Inheritance in Java is a mechanism in which one object acquires all the properties and behaviors of a parent object. It is an important part of OOPs.The idea behind inheritance in Java is that one can create new classes that are built upon existing classes. When you inherit from an existing class you can reuse methods and fields of the parent class. Moreover, you can add new methods and fields in your current class also.Inheritance represents the IS-A relationship which is also known as a parent-child relationship.

    • Types of inheritance There are five types of inheritance.
    1. Single Inheritance - In single inheritance, a single subclass extends from a single superclass. i.e. class Car extends Vehicle.
    2. Multilevel Inheritance - In multilevel inheritance, a subclass extends from a superclass and then the same subclass acts as a superclass for another class.  i.e.class Car extends Vehicle class Vehicle extends Honda class Hond extends HondaCity
    3. Hierarchical Inheritance - In hierarchical inheritance, multiple subclasses extend from a single superclass. i.e. class Car extends Vehicle class Car extends Honda class Car extends HondaCity
    4. Multiple Inheritance - In multiple inheritance, a single subclass extends from multiple superclasses. Java doesn't support multiple inheritance. However, we can achieve multiple inheritance using interfaces.
    5. Hybrid Inheritance - Hybrid inheritance refers to a combination of multiple inheritance and hierarchical inheritance. In Java, multiple inheritance of classes is not directly supported, but it can be achieved using interfaces
